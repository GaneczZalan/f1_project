# F1 Aerodynamic Concept – 2026 FIA Regulations

This project focuses on the aerodynamic development of a Formula 1 car concept based on the 2026 FIA technical regulations. The long-term objective is to build a CFD-based framework capable of analysing and comparing different car concepts under realistic operating conditions.

## Overview

- Full aerodynamic concept developed from 2026 FIA regulations  
- CFD-ready surface model created for aerodynamic evaluation  

## Long-Term Vision

The goal of this project is to develop a consistent simulation framework to:

- Model multiple F1 car concepts across the grid  
- Compare aerodynamic performance between different design philosophies  
- Analyse behaviour under realistic conditions:
  - Ride height variation (aero maps)  
  - Rake sensitivity  
  - Yaw and cornering conditions  (cornering maps)
